# 手配カレンダー (tehai-calendar)

職場での物品購入時に、納期リードタイムを営業日数でひと目で確認できるChrome拡張機能です。新しいタブページをカレンダーに置き換えて表示します。

## 主な機能

- **営業日リードタイム表示** — 土日・祝日・年末年始・カスタム休日を除いた営業日の連番を各日付に表示
- **祝日自動取得** — [holidays-jp API](https://github.com/holidays-jp/api) から日本の祝日を自動取得（30日キャッシュ）
- **年末年始** — 12/29〜1/3 をビルトイン休日として自動適用
- **カスタム休日** — 会社独自の休業日を登録可能（リードタイム計算から除外）
- **納品NG日** — リードタイムには数えるが納品不可の日を登録可能（斜線パターンで表示）
- **毎年繰り返し・範囲指定** — カスタム休日/納品NG日は毎年繰り返しや日付範囲での一括登録に対応
- **製番メモ** — 品名と製番をスタック登録、クリックでコピー
- **手配先メモ** — コードと企業名をスタック登録、クリックでタブ区切りコピー
- **日付コピー** — 日付クリックで YYYYMMDD 形式をクリップボードにコピー
- **マウスホイールナビゲーション** — ホイール操作で前月/翌月を切り替え

## インストール

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

- 新しいタブを開くとカレンダーが表示されます
- 各日付の下の数字が「今日から何営業日目か」を示します
- 拡張機能の設定ページからカスタム休日・納品NG日を管理できます
